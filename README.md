# Applied AI Engineer Portfolio

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

FAANG-style Applied AI Engineer portfolio — production ML systems, RAG pipelines, LLM applications, and backend AI architecture.

## Live Site

Deployed on Vercel with automatic CI/CD from GitHub.

## Featured Systems

| System | Description | Live API |
|--------|-------------|----------|
| [Fraud Risk Analyzer](https://github.com/JRA-CodingLab/fraud-risk-analyzer) | ML fraud detection with SHAP explainability | [API](https://fraud-risk-analyzer-api.onrender.com/docs) |
| [Document QA System](https://github.com/JRA-CodingLab/document-qa-system) | RAG pipeline with source citations | [API](https://document-qa-system.onrender.com/docs) |
| [Hybrid Recommender Engine](https://github.com/JRA-CodingLab/hybrid-recommender-engine) | ALS + association rules recommendation API | [API](https://hybrid-recommender-engine.onrender.com/docs) |
| [News Digest Engine](https://github.com/JRA-CodingLab/news-digest-engine) | LLM-powered news aggregation & summarization | [API](https://news-digest-engine.onrender.com/docs) |

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Language:** TypeScript
- **Hosting:** Vercel
- **CI/CD:** Automatic via GitHub + Vercel integration

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
